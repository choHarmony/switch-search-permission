## 🕹 스위치 (Switch)  
네이버 블로그 글 검색 허용 여부 전환 자동화 프로그램 (Selenium)  
</br>  
### 🛠 사용 방법   
```
1. 프로그램을 실행시킨 후 네이버 계정 정보와 검색 허용 여부를 전환하고 싶은 카테고리의 아이디를 입력해준다.
2. 끝! 원하는 카테고리 속 글의 검색 여부가 자동으로 '허용'으로 변경된다.
```   
</br>
